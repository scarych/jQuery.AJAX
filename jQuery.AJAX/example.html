jQuery AJAX plug-in demo's

Need simple XMLHttpRequest functionality? This should do the trick. I may re-write this in pure java-script later but for the ease of use and quickness of implementation this will do.

Example using default options

The basic usage example can be demonstrated here.

$('#formID').AJAX();

Example setting customized X-ALT-Header value (good for XSRF)

By changing the 'appID' option you can customize the X-ALT-Header that gets sent to the server.

$('#formID').AJAX({appID:'myWickedApp'});

Example changing the context option

Associating the XMLHttpRequest with a different DOM object example.

$('#formID').AJAX({context:'divID'});

Example disabling auto-caching

Simple method of disabling the caching mechanisms.

$('#formID').AJAX({cache:false});

Enable cross-domain requests

This example demonstrates enabling cross-origin requests using JSONP, but supports XML, SCRIPT, JSON and JSONP data types.

$('#formID').AJAX({type:'jsonp'});

Example using an onSuccess callback function

Here you can set a custom function to get executed upon the successful execution per form.

$('#formID').AJAX({callback:function(results){
   console.log(results);
}});

Example using an onError callback function

Here you can set a custom function to get executed upon the errors encountered.

$('#formID').AJAX({errCallback:function(results){
   console.log(results);
}});
